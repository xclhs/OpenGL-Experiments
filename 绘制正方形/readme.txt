开发编译环境：
Code::Blocks [Version-16.01]
FreeGlut[Version 3.0.0]
GLEW[Version 2.1.0]
OpenGL[Version 3.2]
显卡支持最高OpenGL版本4.3

程序功能：
程序主要功能绘制两个三角形从而构成一个四边形，其背景着色为白色，图形着色为红色。

使用说明：
运行code::blocks将所需文件加入项目中，点击debug即可。

实现技术：
实现的技术主要利用顶点输入、顶点着色器、编译着色器、片段着色器、链接顶底属性、索引缓冲对象等。通过利用vec2进行两个三角形的顶点坐标输入，并编译链接两个着色器属性，然后将最终结果绘制到窗口中。
